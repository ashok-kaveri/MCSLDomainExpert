    TrelloClient class:
    ```python
    class TrelloClient:
        BASE = "https://api.trello.com/1"

        def __init__(self, api_key=None, token=None, board_id=None):
            self.api_key  = api_key  or os.getenv("TRELLO_API_KEY", "")
            self.token    = token    or os.getenv("TRELLO_TOKEN",   "")
            self.board_id = board_id or os.getenv("TRELLO_BOARD_ID", "")
            if not all([self.api_key, self.token, self.board_id]):
                raise ValueError(
                    "Trello credentials missing. Set TRELLO_API_KEY, TRELLO_TOKEN, TRELLO_BOARD_ID in .env"
                )

        @property
        def _auth(self) -> dict:
            return {"key": self.api_key, "token": self.token}

        def _get(self, path: str, **params) -> Any:
            r = requests.get(f"{self.BASE}/{path}", params={**self._auth, **params})
            r.raise_for_status()
            return r.json()

        def _post(self, path: str, **data) -> Any:
            r = requests.post(f"{self.BASE}/{path}", params=self._auth, json=data)
            r.raise_for_status()
            return r.json()

        def _put(self, path: str, **data) -> Any:
            r = requests.put(f"{self.BASE}/{path}", params=self._auth, json=data)
            r.raise_for_status()
            return r.json()
    ```

    Methods (all use _get/_post/_put helpers):
    - `get_lists() -> list[TrelloList]` — GET boards/{board_id}/lists, return [TrelloList(id=l["id"], name=l["name"], pos=l.get("pos",0)) for l in data]
    - `get_list_by_name(name: str) -> TrelloList | None` — filter get_lists() by name, return first match or None
    - `create_list(name: str, pos: str = "bottom") -> TrelloList` — POST lists, data={name, idBoard=board_id, pos}
    - `get_board_members() -> list[dict]` — GET boards/{board_id}/members, return list of {id, fullName, username}
    - `get_cards_in_list(list_id: str) -> list[TrelloCard]` — GET lists/{list_id}/cards
    - `create_card_in_list(list_id: str, name: str, desc: str = "", member_ids: list[str] | None = None, list_name: str = "") -> TrelloCard` — POST cards, data={idList=list_id, name, desc, idMembers=member_ids or []}; return TrelloCard from response
    - `move_card_to_list(card_id: str, list_name: str) -> None` — resolve name with get_list_by_name, then _put(f"cards/{card_id}", idList=list.id)
    - `move_card_to_list_by_id(card_id: str, list_id: str) -> None` — DIRECT: _put(f"cards/{card_id}", idList=list_id); log move; no name lookup
    - `add_comment(card_id: str, text: str) -> None` — _post(f"cards/{card_id}/actions/comments", text=text)
    - `update_card_description(card_id: str, new_desc: str) -> None` — _put(f"cards/{card_id}", desc=new_desc)
