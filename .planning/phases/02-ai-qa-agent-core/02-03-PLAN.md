   **click** — 8-strategy fallback chain on `_get_app_frame(page)`:
     1. `frame.get_by_role("button", name=sel, exact=True).first.click()`
     2. `frame.get_by_role("link", name=sel, exact=True).first.click()`
     3. `frame.get_by_label(sel, exact=True).first.click()`
     4. `frame.get_by_text(sel, exact=True).first.click()`
     5. `frame.get_by_text(sel).first.click()` (partial match)
     6. `frame.locator(sel).first.click()` (CSS/xpath)
     7. `frame.locator(sel).first.dispatch_event("click")` (JS dispatch)
     8. Log "click failed all strategies" and return False
     Try each, catch PlaywrightTimeoutError/Exception, move to next.
   
   **fill** — locate input by label or placeholder on `_get_app_frame(page)`:
     1. `frame.get_by_label(action["label"]).fill(action["value"])`
     2. `frame.get_by_placeholder(action.get("label", "")).fill(action["value"])`
     3. `frame.locator(action.get("selector", "input")).first.fill(action["value"])`
   
   **scroll** — `page.mouse.wheel(0, action.get("delta_y", 300))`
   
   **navigate** — MCSL-specific:
     ```python
     url_key = action.get("url", "").lower()
     url_map = _build_url_map(app_base, config.STORE)
     target = url_map.get(url_key, action.get("url", ""))
     page.goto(target, wait_until="domcontentloaded")
     page.wait_for_timeout(500)
     return True
     ```
   
   **switch_tab** — wait for new page event or use context.pages[-1]:

     pages = page.context.pages

     ```

     page.close()

     ```
