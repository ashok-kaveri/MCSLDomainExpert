# Integrating USPS Via EasyPost on Shopify

While there are several ways to utilize USPS for your shipping needs, one of the highly recommended methods is through EasyPost. It makes shipping simpler and more efficient. In this guide, we’ll walk you through setting up USPS with EasyPost on Shopify using the [PH Multi-Carrier Shipping Label App](https://apps.shopify.com/multi-carrier-shipping-label) by PluginHive. This will help you streamline your shipping process and make sure your customers get their orders smoothly.

## Sign Up for EasyPost

To integrate USPS with EasyPost, you first need to create an EasyPost account.

  * Visit the [EasyPost website](https://www.easypost.com/) and click on **Sign up free** to create your account.

* * *

  * Fill in your information accurately and click on **Sign up free.**

* * *

  * You’ll receive a verification email. Enter the code from the email and click **Verify.**

* * *

  * After verifying, you’ll be taken to the login page. Use your email address and password to log in.

* * *

Now, you’re all set to integrate USPS via EasyPost.

* * *

## Get Your EasyPost API Key to Integrate USPS

An API key functions like a password for connecting USPS with your Shopify store via EasyPost. It ensures safe and secure communication for obtaining rates, printing labels, and tracking shipments.

  * To get the API Key, go to the **Account Settings > API Keys**.

* * *

  * Click on **Add Additional API Key**. Select **Production** or **Test** from the dropdown menu based on your requirements. 
    * **Production** : Use this option for real-time(live) shipping purposes.
    * **Test** : Use this option for testing and development purposes.

* * *

**_Note_**  
  
* If the API Key options are missing, please contact EasyPost customer support for assistance.  
  

* * *

Now, you have the API keys to integrate USPS through EasyPost.

* * *

## Add Funds to Your EasyPost Wallet

Adding funds to your EasyPost wallet ensures you can pay for shipping labels, insurance, and other services without any delays. This keeps your shipping process smooth and uninterrupted, as you won’t face any payment issues when printing USPS shipping labels.

  * To add funds, go to **Account Settings > Billing > Setup Wallet.**

* * *

Now, your EasyPost setup is complete and ready for use for USPS shipping.

* * *

## Set Up USPS with EasyPost on Your Shopify Store

To connect USPS to your Shopify store, install the **[Multi-Carrier Shipping Label App](https://apps.shopify.com/multi-carrier-shipping-label)**. The app makes shipping easy by letting you print labels directly from Shopify and track orders, keeping everything simple and smooth.

  * Go to the **Multi-Carrier Shipping Label App Settings > Carrier (+) > EasyPost.**

* * *

  * Enter your API keys.
  * Use **Production API Keys** for real-time shipping.

* * *

  * Click on **Add**.

* * *

The USPS will be connected to your Shopify app automatically.

* * *

## Configure USPS Shipping Rates

To display USPS shipping rates at checkout, you need to set up a rate automation rule in the Shopify shipping app. This rule will be used to display real-time USPS shipping rates.

  * To set up a rate automation rule, navigate to **Settings > Shipping Rates > Rate Automation**.

* * *

  * Select **Add Rule**

  * Name the rule and select the USPS services you need.
  * Then click **Add Rule.**

* * *

As the next step, you need to add a rule for generating USPS labels in the app. This rule will be used for auto-label generation.

  * Go to **Settings > Automation > Setup**.

* * *

  * Click on **Add New** and create a new rule for generating USPS labels in the app.

* * *

  * Give a name to the rule and choose the USPS services you want.
  * Then, click **Add Rule.**

* * *

Now that the rules are set up for showing USPS rates and generating shipping labels, you can display USPS shipping rates at checkout and start shipping orders.
